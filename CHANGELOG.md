# Changelog

All notable changes to this project will be documented in this file.

## [0.7.0] - 2024-05-22

### Added
- **Macro System**: Create and run sequences of tools (Tool Chains) for automation.
- **Floating Action Button (FAB)**: Quick access button that stays on top of the page.
- **Native Context Menu**: Right-click "Send to MCP SuperAssistant" integration.
- **Voice Input**: Speech-to-text support in the chat input.

## [0.6.2] - 2024-05-22

### Added
- **Quick Actions Command Palette**: Press `Cmd+K` (or `Ctrl+K`).
- **Smart Context Integration**: Select text on any webpage and import it.
- **Onboarding Tour**: Guided introduction for new users.
- **Auto-Retry**: Robust connection recovery.

## [0.6.0] - 2024-05-22

### Added
- **Analytics Dashboard**: Usage metrics.
- **Rich Renderer**: Visualizes tool outputs.
- **Keyboard Shortcuts**: Global hotkeys.
- **Auto-Execute Whitelist**: Safe mode.
